# Privacy Policy for Deep Seek Tone Changer

**Last updated: March 3, 2026**

## Data Collection
Deep Seek Tone Changer does not collect, store, or transmit any personal data, user information, or browsing history.

## Permissions
- **Host Permission (chat.deepseek.com)**: Required solely to insert prompt text into the DeepSeek chat input field. No data is read, stored, or sent externally.
- **Scripting**: Used only to modify the chat input field locally on your device.

## Third-Party Services
This extension does not use any third-party analytics, tracking, or external servers. All operations are performed locally on your device.

## Changes
No changes to data handling are planned. If this policy changes, the extension will be updated accordingly.

## Contact
For questions, refer to the Chrome Web Store listing.
